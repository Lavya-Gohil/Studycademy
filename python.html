Chapter 1: Introduction to Python

Welcome to Python Programming! In this chapter, you will learn how to set up Python on your computer, write your first program, and understand the basics of Python syntax.

Topics Covered

Setting up Python, Writing your first Python program, Basic Python syntax

Setting up Python

To set up Python on your computer, visit the official Python website and download the installer for your operating system.

Writing Your First Program

Open a text editor or IDE and write the following code:

print("Hello, World!")

Save the file with a .py extension and run it using the Python interpreter by typing python filename.py in the command line.

Basic Python Syntax

Understanding the basic syntax of Python is essential:

Variables: Used to store data values.

x = 5; name = "Alice"

Data Types: Common types include (int, float, str, list, dict).

Operators: Includes arithmetic (+,-,*,/), comparison (==, !=, >, <), and logical (nd, or, not) operators.

Control Structures:


        # If Statements
        if x > 0:

        else:

Functions

Functions are defined using the def keyword:

def greet(name):
            return f"Hello, {name}!"

Modules and Packages

Modules and packages are essential for organizing and reusing code:

Modules: Files containing Python code that can be imported and used in other scripts.

import math
        print(math.sqrt(16))

Packages: Directories containing multiple modules. Use pip to install packages from the Python Package Index (PyPI).

Error Handling

Use try and except blocks to handle exceptions:

try:
            result = 10 / 0
        except ZeroDivisionError:
            print("Cannot divide by zero")

Chapter: Data Types and Variables

Understanding data types and variables is fundamental in programming. In this chapter, you will learn about the different data types in Python and how to use variables to store data.

Topics Covered

What are Data Types?, Common Data Types in Python, Variables, Variable Naming Rules, Type Conversion

What are Data Types?

Data types are the classification of data items. They tell the interpreter or the compiler how the programmer intends to use the data.

Common Data Types in Python

Python supports several standard data types:

Integers: Whole numbers without a fractional part.

x = 10

Float: Numbers with a fractional part.

y = 3.14

String: Sequence of characters.

name = "Alice"

List: Ordered collection of items.

fruits = ["apple", "banana", "cherry"]

Tuple: Ordered collection of items that is immutable.

coordinates = (10, 20)

Dictionary: Collection of key-value pairs.

person = {"name": "Alice", "age": 25}

Boolean: Represents one of two values: True or False.

is_student = True

Variables

Variables are used to store data that can be referenced and manipulated in a program. They have a name and hold a value.

Variable Naming Rules

When naming variables in Python, follow these rules:

Variable names can contain letters, numbers, and underscores. Variable names must start with a letter or an underscore (_). Variable names are case-sensitive. Variable names cannot be Python keywords.

Type Conversion

Type conversion is the process of converting a value from one data type to another. Python provides several functions for this:

int(): Converts a value to an integer.

number = int("10")

float(): Converts a value to a float.

pi = float("3.14")

str(): Converts a value to a string.

text = str(10)

Topics Covered

If Statements

Conditional statements allow you to execute different code blocks based on conditions. Here's an example:


x = 10
if x > 5:
    print("x is greater than 5")
else:
    print("x is not greater than 5")

For Loops

Loops allow you to repeat code multiple times. A for loop example:


for i in range(5):
    print(i)
